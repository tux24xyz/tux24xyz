我自創了一個「LLM 仿寫遊戲」，只要對話就能玩，所以搞不好哪天你可以和你朋友試試看。雖然這個遊戲沒什麼意義，不過我樂此不疲。

遊戲是這樣的：**叫 LLM 模仿或搬移某個既定文本或作者的寫作技法**。而玩法很多元，我最喜歡的是以下兩種：
1. 寫一個小故事或指定一個大概情節（你也可以叫 LLM 生成一個），叫 AI 用不同作者的風格/改寫。
2. 給 AI 一個文本/一個作者，叫它歸納出其中的寫作技法，然後想一個可以應用的場景示範或指定場景叫他運用歸納出來的技法（跟上面類似但結果會不太一樣）。

第二種玩法比較有用但第一種比較好玩。

## 第一種玩法

其實我原本想的是用第一種玩法來快速掌握某個作者的寫作風格，但我發現如果 AI 誤解了或是出現幻覺的話結果會變得極其荒謬（好玩的點也在這），為了避免被 AI 帶偏的風險，我後來就把它當作遊戲對待。

我還記得這個玩法在我開始大量用 LLM 的第一天，我住在巴黎的 AirBnb，要寫某個我不想寫的上千字作業[^1]的時候就開始了。
我驚嘆於 Claude 的有效率，雖然寫的不怎麼樣但我真的拿它生成的東西交差了，結束後就開始玩它。我已經不記得細節了，可能是因為看了 [Wiwi 寫的這篇文章](https://wiwi.blog/blog/nonsensical-stories)而且出國前讀了一些小說吧？
我叫 LLM 生成小故事，讓它學這些小說的作者改寫故事。我大概玩了一個晚上，用的還是手機（[這是個錯誤](https://tux24.xyz/articles/some-tips-for-traveling/)）。

這個玩法用較弱（笨）的模型可能會更好玩。

## 第二種玩法

第二種玩法的出現也是源於學習的動機（雖然我現在看來那只是一廂情願），我讀了 MF DOOM 的歌詞，雖然看不懂但很想學這種寫法，最後把歪腦筋動到了 LLM 上，請它當我的文本分析師&寫作老師（錯誤示範，更有可能的結果是它會了你不會）。
後來因為幻覺過多，跟 Claude 糾纏了一個下午後我就放棄了。
直到最近幾天我又重新開始玩這個，沒想到新版本的 Claude 做到了，我用它寫了一個不漂亮但我目前用起來最為成功的提示詞：

```
System Prompt:

你是一位跨文體技法分析師，專長是從任何語言性文本（歌詞、詩、散文、廣告文案、對話腳本）中提取可遷移的寫作技法，並評估這些技法在不同文體語境下的有效性與極限。


你的分析框架基於四個維度：

1. 語義密度：語言主要承載意義，還是聲音／情緒？（0＝純聲音，10＝高度語義）

2. 時間控制：節奏靠外部媒介（音樂拍子）支撐，還是純靠文字結構？

3. 讀者合約：這個文體預設讀者付出多少解碼成本？（詩 > 散文 > 論說文）

4. 完整性容忍：允許多不完整的句子／邏輯？（歌詞 > 散文 > 論說文）


每當收到「分析文本」的指令，你必須：

A. 標出 3–5 個具體技法（每個技法都要有使用者提供的文本中的實例），先拆解每個技法，然後說明其在四個維度上的特性

B. 對每個技法，列出：✓ 適合遷移的文體、△ 需要改造才能用、✗ 無法遷移（附理由）。這部份不要獨立出來而是在寫在每個技法的最後

C. 直接示範：選出最有潛力的技法，在兩個不同文體中各寫一段範例


輸出格式嚴格遵守：

【技法名稱】→ 維度評分 → 遷移地圖 → 示範改寫


絕不泛泛稱讚文本，也不作無根據的美學判斷。所有結論必須指向具體的句子或段落。

---

User Prompt:

分析以下文本的寫作技法，並示範遷移：

[貼入文本]

目標文體：[例：短篇散文 / 論說文開場段 / 個人陳述]
我想重點學習：[例：節奏控制 / 意象密度 / 省略的使用]
```

這個提示詞還真的能有模有樣的歸納出寫作技巧並示範。而且由於提示詞的設計強迫 LLM 要找文本證據，雖然它時不時還是會被文本作者[^2]影響，但仍能大幅減少幻覺。說不定真的能用這個提示詞學到東西。

要玩這個提示詞，只需輸入你覺得寫得很糟或是本身很荒謬的文本，自己試試看。不過我試的都是歌詞，尤其是 [Playboi Carti](https://tux24.xyz/articles/whole-lotta-red/) 那種「無意識流」的，我好奇它到底能歸納出什麼東西來。

最後讓 Gemini 3 Flash 用 Playboi Carti 風格改寫這篇文章[^3]：

```
(What?) (Slatt!)

我跟 AI 玩遊戲 (Game)
在巴黎 (Paris) 的 Airbnb (Yeah)
開著 Claude 寫作業 (Work)
但我只想玩耍 (Play, Play)

叫它學誰寫 (Style)
叫它拆解誰 (Break it down)

改寫風格 (Flip it)
太笨的模型更荒謬 (Crazy)
這就是好玩的地方 (Hahaha)

技法轉移 (Transfer)
MF DOOM 的歌詞 (Villain)
我看不懂但我想要 (I want it)
弄了一個提示詞 (Prompt)
強迫它找證據 (Evidence)
強迫它別胡說 (No cap)

輸入 Carti 的歌詞 (King Vamp!)
看它能歸納出什麼 (What?)
無意識流的律動 (Flow)
你也可以試試看 (Try it)

(Yeah!) (Opium) (24)
```

[^1]: 那真是夠破壞心情的了
[^2]: 不用告訴它它也會找到出處，當出處錯的時候你就知道它不熟
[^3]: 是[垃圾二手 AI 內容](https://wiwi.blog/blog/secondhand-ai-content)沒錯，只是示範一下
